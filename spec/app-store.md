# Long Description

Verdict is a social voting platform where people post relationship situations and receive a verdict from the community.

How it works:

1. Create a case
2. Describe the disagreement
3. Ask your question
4. Community votes
5. Receive your verdict

Features:

- Anonymous posting
- Vote before results
- Relationship categories
- Community verdicts
- Save and share cases
- Premium insights
- No comments
- No direct messaging
